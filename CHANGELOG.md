# Change Log

All notable changes to the "md-to-pixiv-novel" extension will be documented in this file.

## [1.0.2] - 2026-02-09

### Added

- 熟語ルビ記法（`{熟語|ルビ1|ルビ2|...}`）に対応
  - 親文字とルビの数が一致する場合、個別のルビに分割して変換
  - 例: `{電子出版|でん|し|しゅっ|ぱん}` → `[[rb:電 > でん]][[rb:子 > し]][[rb:出 > しゅっ]][[rb:版 > ぱん]]`

### Changed

- READMEに熟語ルビの説明を追加
- ライセンス情報をWTFPLに更新

## [1.0.1] - 2026-02-08

### Fixed

- 画像タグのIDにハイフン付きページ番号（例：`137802828-3`）が含まれる場合の変換に対応
- 日本語altテキスト（例：`![手のひらにブローチ](ID)`）を正しく変換できるよう修正

## [1.0.0] - 2026-02-04

### Added

- ファイル全体を変換するコマンドを追加
- 選択範囲を変換するコマンドを追加
- 見出し（h2〜h6）を章タイトルタグに変換する機能を追加
- 改ページ（`===`）を`[newpage]`タグに変換する機能を追加
- 画像タグを`[pixivimage:ID]`タグに変換する機能を追加
- ページジャンプ（`[%N]`）を`[jump:N]`タグに変換する機能を追加
- URLリンクを`[[jumpuri:テキスト > URL]]`タグに変換する機能を追加
- ルビ記法（`{親文字|ルビ}`）を`[[rb:親文字 > ルビ]]`タグに変換する機能を追加
- 右クリックメニューからの実行をサポート
- 日本語UIメッセージを提供

### Security

- 初期リリース
